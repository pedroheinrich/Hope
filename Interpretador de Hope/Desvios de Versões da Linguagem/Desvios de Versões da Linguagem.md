
# :computer: Desvios de outras versões do idioma :computer:

Os seguintes recursos são especiais para esta versão do **HOPE**:

- Padrões irrefutáveis
- Seções de operadores
- Funções de mais de um argumento
- Definições de tipos recursivos
- `input`
- `read`
- `write`
- `private`
- `abstype`
- `letrec`
- `whererec`

Outras versões podem não suportar avaliação lenta completa e podem ter vários recursos não fornecidos aqui.

Esta versão também oferece suporte a construtores de dados e tipo curry.
