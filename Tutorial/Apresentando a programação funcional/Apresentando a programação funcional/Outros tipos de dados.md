# Outros tipos de dados

Hope fornece dois outros tipos de dados primitivos. Um `truval` (valor verdadeiro) é equivalente a um booleano Pascal e possui valores `true` e `false`. Já vimos a expressão `x > y` definindo um valor verdade. `>` é uma função padrão cujo tipo é `num # num -> truval`. Podemos usar valores verdade em expressões condicionais e combiná-los com as funções padrão `e`, `ou` e `não`.

Caracteres únicos são do tipo `char`, com valores `'a'`, `'b'` e assim por diante. Os caracteres são mais úteis como componentes de estruturas de dados, como cadeias de caracteres.
